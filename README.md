## 📌 Project Explanation
- 고민되는 여행 코스를 Chat Gpt에게 추천 받을 수 있습니다.
- 여행 후기를 다른 사람들과 공유할 수 있습니다.
- 여행지에서 주변 검색으로 필요한 장소를 쉽게 찾을 수 있습니다.
- Git hub Actions를 이용해 push시 자동 배포하도록 설정하였습니다.
- 검색/추천 받은 여행지를 내 페이지에서 조회할 수 있습니다.
